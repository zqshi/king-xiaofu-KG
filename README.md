# IM风格智能助手

基于IM（即时通讯）交互模式的企业级智能助手平台，采用二分栏布局设计，集成文件处理、智能Agent响应与自定义指令管理三大核心功能。

## 功能特点

### 1. 文件上传与处理
- 支持拖拽上传和多文件选择
- 文件上传进度实时显示
- 智能检测文件状态（新增/已存在/相似）
- 管理员可处理异常文件（覆盖/新版本/标记重复/合并）

### 2. 智能Agent交互
- IM风格对话气泡界面
- 实时状态反馈（处理中/完成/异常）
- 支持文本消息和文件交互
- 管理员权限控制和操作面板

### 3. 自定义指令管理
- 左侧栏指令分类与搜索
- 支持创建和编辑自定义指令
- 指令模板支持变量占位符
- 权限设置（公开/私有）

## 技术实现

- **前端框架**：HTML5 + CSS3 + JavaScript
- **UI框架**：Tailwind CSS v3
- **图标库**：Font Awesome
- **动画效果**：CSS动画 + 过渡效果

## 如何使用

1. 克隆项目到本地
2. 直接打开 `index.html` 文件即可运行
3. 无需安装任何依赖，所有资源通过CDN加载
